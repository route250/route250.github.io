# あみだくじ プライバシーポリシー

最終更新: 2026-06-22

## 基本方針

「あみだくじ」は、ユーザーのプライバシーを尊重し、アプリの利用に必要な情報を端末内で扱います。

## 収集する情報

現行バージョンでは、開発者がアプリを通じてユーザーの個人情報、利用状況、端末情報、広告識別子、位置情報を収集することはありません。

## 端末内に保存される情報

アプリの機能として、以下の情報がユーザーの端末内に保存されます。

- 作成したくじの項目
- 番号ごとの名前
- くじの結果
- メモ
- 履歴

これらの情報は、アプリ内で履歴を表示するために使用されます。外部サーバーへの送信、第三者への提供、広告目的での利用は行いません。

## 第三者サービス

現行バージョンでは、広告、アクセス解析、外部ログイン、クラウド同期、外部データ送信を行う第三者サービスは使用していません。
