EyeChecker

音声で遊べる簡易視力チェックアプリです。上・下・右・左を声で答えて、参考値としての結果を表示します。

このアプリについて

  • ランドルト環風の向きを音声で回答します。
  • 結果は簡易的な参考値です。
  • 医療用途、診断、視力証明、運転や資格確認には使用できません。

注意事項

EyeCheckerは医療機器ではありません。見えにくさや目の不調が続く場合は、眼科などの医療機関に相談してください。

利用する権限

音声入力のために、マイクと音声認識を使用します。マイクまたは音声認識を利用できない場合、検査を開始できません。
